Aspen Software Foundation

An independent organization building, maintaining, and supporting free and open source software — from kernel development to hosting infrastructure, all self-operated on ASF-owned hardware.

What We Do

We develop systems-level software, provide infrastructure for open source projects, and run a fully self-hosted development environment. No third-party platforms, no vendor lock-in. Everything under aspensoftwarefoundation.com is built and operated by us.
