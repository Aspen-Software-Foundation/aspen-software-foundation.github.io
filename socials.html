Get Involved

The Aspen Software Foundation is community-driven. Whether you write code, documentation, or just want to follow along, there's a place for you.

Community Channels

Discord https://discord.gg/SBeSJKyjEp Real-time chat, support, and collaboration. Most active channel.
IRC #aspensoftwarefoundation @ libera.chat For those who prefer IRC.
Forums forum.aspensoftwarefoundation.com Long-form discussion and project support.
Wiki wiki.aspensoftwarefoundation.com Documentation, guides, and project references.

Contributing

All ASF projects are open source and accept contributions. Source code is at git.aspensoftwarefoundation.com, with mirrors on GitHub and Codeberg.

Code Browse repositories, pick an issue, submit a patch. All projects are under open licenses.
Documentation Improve project docs, write guides, or expand the wiki.
Design UI/UX contributions welcome on projects with frontend components.
Bug Reports Open issues on the relevant repository at git.aspensoftwarefoundation.com.
